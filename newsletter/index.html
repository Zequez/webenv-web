2nd of November 2025

The struggle to explain what has never existed

Today marks the first update report and so I'm going to wing it.

Over the years I've been inching closer and closer to the ultimate truth of life and everything. Or so I believe. I have not found it yet, but in the meantime, I've been also creating prototypes that have brought me to what I'm now calling the Web Environment.

The web has always been kind of a hobbie for me, and a creative medium━━

I've decided I'm just going to share with you what I've been working on rather than go into a long dramatic adventure about the story of my life, how I got here and and how I'm such a victim of my own genius, and the pain of being misunderstood.

I'm making a list of the most interesting things I've created and then expanding on them:

  • The visual Electron-based UI for managing projects, version controlling with GIT, syncing to remote, selecting substrate, awakening the dev-server-process, cloning, building and publishing.
  • The very beautiful multi-process architecture
  • Parent-repo architecture
  • What I'm calling the toroidal-navigation-pattern for building websites
  • WebEnv-based websites for my neighbourhood and the regional agroecologic network using toroidal navigation
  • Experimenting with scraping Startover 700+ websites and displaying them using toroidal-navingation
  • Video overview of the Web Environment
  • First experiment with canvas-based mainframe UI
  • First experiments with Svelte AST
  • First experiments with HTML blocks editing
  • Infinite canvas substrate and publishable website
  • TypeScript simple browser-to-system tunneling server
  • Soft internationalization approach for multi-language people
  • First experiments with self-exported integrated TLDraw component

Looking back, I've been having a lot of fun.
